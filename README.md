# Ease
An AI-ready productivity platform designed for students to manage schedules, assignments, study sessions, and academic deadlines in one place.

## About
EASE aims to bring together task management, scheduling, AI assistance, notes, and study tools into one application.
> This repository documents the complete journey of building EASE from scratch.
